Install, wrap, observe.

Use CutCtx with a workflow you already have, then use your own telemetry and savings report to decide whether to scale.

1. Install CutCtx

pip install "cutctx-ai[all]"

2. Initialize your local setup

cutctx init

3. Wrap a supported agent

cutctx wrap codex

Replace codex with the supported agent you use, such as Claude Code, Cursor, Cline, Continue, Aider, OpenHands, Windsurf, Zed, OpenCode, OpenClaw, or Goose.

4. Review your own result

cutctx savings report

For provider configuration, Docker, Kubernetes, MCP, and deployment options, use the project documentation.
