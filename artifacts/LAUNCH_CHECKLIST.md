# Headroom Launch Checklist

## Positioning

- finalize the one-line product description
- align homepage, enterprise page, and sales copy
- remove stale roadmap claims from launch materials

## Packaging

- approve pricing
- approve packaging matrix
- approve support tiers

## Trust

- publish security one-pager
- publish security and privacy doc
- prepare deployment packet
- prepare procurement packet

## Sales Readiness

- finalize demo environment
- finalize discovery questions
- finalize pilot offer
- finalize objection handling

## Customer Success

- finalize onboarding runbook
- assign support owner
- define escalation path

## Launch Exit Criteria

- one pricing source of truth
- one packaging source of truth
- one security packet source of truth
- one onboarding runbook
- one pilot flow
