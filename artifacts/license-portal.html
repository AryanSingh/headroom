Quick Start

Install Cutctx via pip and run locally

pip install "cutctx-ai[all]" cutctx proxy --port 8787
Your proxy will be available at http://localhost:8787. Set your API keys and license key in environment variables.

Configure a client to route traffic through Cutctx

export OPENAI_API_BASE=http://localhost:8787/v1 export ANTHROPIC_API_KEY=sk-ant-... export CUTCTX_LICENSE_KEY=team-...
Your client will automatically compress requests before forwarding to upstream providers. Cutctx supports Anthropic, OpenAI, Google, Bedrock, and Vertex.

Deploy Cutctx in Docker

docker compose up -d
Clone the repository and run docker compose up -d in the project directory. Mount your license key via environment variables or volumes.

Deploy Cutctx on Kubernetes

kubectl apply -f k8s/ # or use Helm helm install cutctx ./helm/cutctx \ --set licenseKey=team-... \ --set hmacSecret=...
For Enterprise deployments, refer to our Kubernetes manifests and Helm chart documentation. Contact hello@cutctx.dev for deployment guidance.
