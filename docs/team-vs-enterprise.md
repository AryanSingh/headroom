## What Both Include

- Core compression pipeline
- CCR reversible retrieval
- Proxy, SDK, CLI, and MCP modes
- Local dashboard
- Multi-provider support
- Docker deployment

## Team Tier

Team is the first paid commercial tier. It is best when one engineering team needs shared visibility and lightweight governance without full enterprise procurement.

Team includes:
- Shared analytics
- Usage and savings reporting
- Policy presets
- Budget controls
- Business-hours support

## Enterprise Tier

Enterprise is for organizations that need the deployment to survive security review and operate as platform infrastructure.

Enterprise includes:
- SSO-aware admin authentication
- Role-based access control
- Audit log query and export
- Retention controls
- Fleet management APIs
- SCIM-style provisioning APIs
- Security review support
- Air-gap deployment path

## Important Note

Business sits between Team and Enterprise. If you need workspace segmentation, project management, historical reporting, exports, Kubernetes standardization, or Helm packaging, use the Business tier before jumping straight to Enterprise for identity or compliance reasons.
