The Problem

Agent context is expensive and ungoverned

Your AI agents burn tokens on verbose tool outputs, logs, and search results. Provider caching helps — but not enough.

💸

Tool outputs bloat fast

Code search results, file diffs, test output, and logs consume 60–90% of your context window — and your LLM budget.

🔒

Provider caching is provider-locked

Native caching from Anthropic, OpenAI, and Google works inside a single provider. It doesn't help across providers, tools, or payload types.

📊

No visibility into AI spend

Teams have no idea where tokens go. No per-team analytics, no project rollups, no exportable reports for ROI reviews.

🛡️

Governance gap

No policy control, no audit trail, no SSO. Enterprise security teams can't approve deployments without these.

How It Works

Three steps to governed, optimized context

Deploy the proxy

Run CutCtx locally, in Docker, or in your Kubernetes cluster. Zero code changes — just point your agent at the proxy.

Context flows through

CutCtx detects content type (JSON, code, logs, diffs) and applies the right compressor. Originals stored locally for retrieval.

Save, govern, measure

60–95% fewer tokens. Team analytics. Policy presets. Audit logs. Your data never leaves your infrastructure.

Security & Privacy

Your data stays in your infrastructure

CutCtx is local-first by default. No prompts, tool outputs, or conversation content ever leave your network.

🏠

Local-first deployment

Runs in your infrastructure, behind your firewall. Docker, docker-compose, or Kubernetes.

🚫

No prompt content in telemetry

Optional telemetry sends only aggregate counts — request count, tokens saved, model distribution. Never message content.

📴

Air-gap support

Pre-download models and run fully offline. No external network access required after initial setup.

🔐

Internal header stripping

Drops all x-headroom-* headers before forwarding to prevent proxy fingerprinting.

⏱️

7-day offline grace period

If license server is unreachable, cached license data is used for up to 7 days. Proxy continues normally.
