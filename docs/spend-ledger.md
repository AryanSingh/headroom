# Spend Ledger

The Spend Ledger provides real-time tracking of token consumption and associated monetary costs across your enterprise deployment.

## Features
- **Asynchronous Ingestion**: The proxy emits spend events out-of-band to ensure zero latency impact on request handling.
- **Budget Alerts**: Configure thresholds to trigger warnings or block further usage when budgets are exceeded.
- **Granular Tracking**: Drill down spend by organization, workspace, model, or individual agent.
- **CSV Exports**: Export raw event data for reconciliation with billing providers.

## Architecture
Spend events are generated by `crates/cutctx-proxy/src/spend/emitter.rs` and sent to the `cutctx_ee/ledger/api.py` ingestion endpoint. Data is stored locally in an SQLite-backed ledger or synced to an external warehouse.
