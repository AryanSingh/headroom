# Cutctx Documentation

## Overview

Cutctx is an open-source context compression layer for AI agents. It reduces token usage by 60-95% while maintaining data integrity through reversible compression.

## Section 1: Feature Description

This section describes feature 1 of Cutctx. The feature provides compression capabilities that reduce token consumption for AI agent interactions. Key benefits include cost reduction, latency improvement, and context window optimization. The implementation uses advanced algorithms including SmartCrusher for JSON compression, CodeCompressor for source code, and DiffCompressor for diff/patch content.

### Usage

```bash
cutctx proxy --port 8787 --compression-mode live-zone
```

### Configuration

| Setting | Default | Description |
|---------|---------|-------------|
| `max_body_mb` | 50 | Maximum request body size |
| `compression_mode` | live-zone | Compression strategy |
